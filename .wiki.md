Project Summary
“Shadow Garden” is an innovative anime streaming web application that offers users a vast library of anime content. The platform leverages advanced AI capabilities for personalized recommendations and fosters a vibrant community through social networking features. Users can engage in discussions, share watchlists, and enjoy a seamless streaming experience.

Project Module Description
Core Features: Streaming anime via the Aniwatch API, a dark red/gothic theme, and smooth animations.
Social Platform: “Otakuverse” for user interactions, posts, and watchlist sharing.
AI Features: Image-to-anime search, waifu search, and an AI chatbot named “Shadow”.
User System: Login/registration, user profiles, watchlist tracking, and continue watching functionality.
Player Features: Advanced video controls, intro/outro skip, and server switching.
Navigation: Redesigned bottom navigation bar for improved accessibility and mobile-friendliness.
